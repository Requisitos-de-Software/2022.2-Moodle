    Introspecção

    1. Definição

       Introspecção se baseia em imaginar que tipo de sistema eu iria querer se eu estivesse executando esta tarefa, utilizando este equipamento, entre outros. Ou seja, imaginar que propriedades um sistema deveria possuir de forma a obter sucesso.

    2. Metodologia

       Um dos participantes do grupo se disponibilizou para fazer sua instropecção e documentar os resultados
