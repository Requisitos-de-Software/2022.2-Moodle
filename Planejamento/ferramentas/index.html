    Ferramentas

    1. Introdução

       No desenvolvimento de um projeto de software uma grande varieade de ferramentas possibilitam um desenvolvimento mais ágil e eficiente. Este artefato busca sintetizar as ferramentas utilizadas pelos integrantes do time.

    2. Ferramentas utilizadas

    Ferramentas Descrição
    GitHub - Plataforma de versionamento em que iremos armazenar todos os arquivos relacionados ao projeto
    Telegram - Ferramenta para a comunicação rápida entre os integrantes
    Vscode - Editor de código destinado ao desenvolvimento de aplicações com controle Git incorporado
    Google Docs - Ferramenta de editor de texto, para a documentação do projeto
    Google Sheets - Ferramenta de criação de planilhas, gráficos e para a documentação do projeto
    Microsoft Teams - Plataforma que vamos usar para gravar a apresentação do projeto
    Discord - Plataforma em que vamos nos comunicar por chamada de áudio
    Lucidchart - Ferramenta case, onde vamos fazer nosso diagrama
    Google Forms - Ferramento do google, para fazer a realização da pesquisa
    Youtube - Para Guardar os videos
    OBS Studio -Para gravação de entrevistas e das apresentações
    Figma - Ferramenta online de prototipagem
    Mondays - Ferramenta de gestão de projeto, para organizar o quadro kanban
    This person does not exist - Ferramenta online para obter as imagens de perfil das personas
    Xtensio - Ferramenta online de design
