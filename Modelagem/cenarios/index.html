    Cenários

    1. Introdução

    Cenários são considerados descrições evolutivas de situações num ambiente, sendo compostos por um conjunto ordenado de interações entre seus participantes. Os mesmos têm sido amplamente utilizados no processo de elicitação dos requisitos de sistemas de software. Neste contexto, cenários são utilizados para descrever as situações de uso do sistema pelos seus usuários e os relacionamentos entre o sistema em desenvolvimento e outros sistemas externos, auxiliando no entendimento e na descoberta de novos requisitos.

    1.1 Modelos de cenário

    Existem cinco formas para a descrição de cenários: * Texto narrativo * Texto estruturado * Diagramas * Imagens * Animações ou simulações

    2.Cenários

    Cenário 1

    CENÁRIO 01
    Título: - Entrega de tarefa
    Objetivo: Entregar um exercício na platforma
    Contexto: Quando o professor deixa exercícios
    Recursos: Documento que contém as respostas dos exercícios
    Atores: Alunos/Usuário
    Episódios: - Usuário faz login na plataforma.
    - Usuário abre o painel da plataforma
    - Usuário seleciona a tarefa para enviar
    - Usuário clica no botão de enviar a tarefa
    - Usuário faz upload do documento
    - Usuário clica em submeter para confirmar o envio do exercício.

    Cenário 2

    CENÁRIO 02
    Título: - Download do material do professor
    Objetivo: - Realizar o download do material disponibilizado pelo professor
    Contexto: Quando Usuário pretende estudar um determinado tópico
    Recursos: Nenhum
    Atores: Alunos/Usuário
    Episódios: - Usuário faz login na plataforma;
    - Usuário abre o painel da plataforma
    - Usuário seleciona a matéria;
    - Usuário seleciona o tópico ou o módulo da matéria
    - O Usuário clica no material para realizar o download

    Cenário 3

    Cenário 03
    Objetivo - Escolher a sua universidade
    Contexto - Local: Logo ao iniciar o aplicativo pela primeira vez
    - Tempo: 2 min
    - Pré-condição: Aplicativo aberto, acesso à internet
    Atores - Alunos/Usuário
    Recursos - Smartphone, internet e o aplicativo
    Exceção - Smartphone para de funcionar
    - Sem acesso à internet
    Episódios - Usuário abre o aplicativo
    - Usuário copia o link do site de atividades de sua universidade
    - Usuário cola o link do site de atividades de sua universidade.
    - Usuário preenche a tela de login com sua identificação

    - Usuário digita a senha de acesso
